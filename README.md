### hey, I'm Ayush 👋

CS undergrad at IIIT Hyderabad. I like taking software that feels like magic and rebuilding it from scratch until it isn't: transport protocols, file systems, kernels.

Currently deep in OS internals and distributed systems, interning at ConnectMyWorld (on-device ML), and setting CTF challenges for my college's hacking club. Sometimes I play them too (2nd place open, Deccan CTF).

**things I rebuilt to understand them**

- [Reliable-UDP](https://github.com/BiscBoi28/Reliable-UDP): TCP-style reliability on raw UDP sockets, in C. Survives 20% injected packet loss with matching MD5s.
- [CoWrite-file-system](https://github.com/BiscBoi28/CoWrite-file-system): a collaborative document store with sentence-level locks, replication and automatic failover.
- [XV6-Enhancements](https://github.com/BiscBoi28/XV6-Enhancements): demand paging with swap, plus four swappable CPU schedulers for xv6-riscv.

I built [small interactive simulators](https://biscboi28.github.io) of these for fun, so you can drop packets and crash servers in the browser instead of reading about it.
